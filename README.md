## jQuery AJAX Tutorial
Yet another jQuery AJAX example, written in Node.js/Express. 

#### Prerequisites
- Node.js
- NPM
- Bower


#### How to use
- Clone this repo
- Execute following commands to install all required dependencies, 
```
# First time only
npm install
cd public 
bower install
cd ..

# Executing
npm start
```
- Access http://localhost:8080 in your browser
